## Backend with NodeJS and Express

Node - JS Runtime

NPM - Package Manager

package.json - project ka config file hai

dependecies - jo cheezein project mein use ho rahi hain

devdependecies - jo cheezein development ke time use ho rahi hain

npm install package-name --save = Dependecies

Example:
npm install express --save
npm install express
npm i express

npm install package-name --save-dev = DevDependecies

Example:
npm install nodemon --save-dev
npm install nodemon -D
npm i nodemon -D

npm remove package-name --save = Remove from Dependecies
npm remove package-name --save-dev = Remove from DevDependecies

npm uninstall package-name --save
npm uninstall package-name --save-dev
